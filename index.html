Work Clock

A macOS menubar app to display time for a different timezone

Work Clock displays the time for another timezone in macOS menubar. It's handy when you work remotely from a different timezone and need to keep track of the work clock when scheduling meetings or discuss time in messages or calls.

Features

  • Display time in 12-hour or 24-hour format
  • Show/hide seconds
  • Select a timezone from a list of 40+ timezones covering the whole world

Contact me on Twitter @jabranr for any questions.

Work Clock requires macOS 10.12 or later. Work Clock is a one-time purchase with all the updates/upgrades forever.

Story

Recently it became difficult for me to work out the correct UK time while I was working remotely from Pakistan. I tried a few apps and macOS clock widget however none of those were satisfactory. I had to follow a few steps and clicks in order to view the current time for the UK.

So I created this app to solve the problem and also learned macOS development. I hope this helps other people with a similar problem.
