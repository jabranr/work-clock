Work Clock

A macOS menu bar app to display time for a different timezone

Work Clock displays time for another timezone i.e. work timezone in case of a remote work.

With remote work becoming a norm, it can be difficult to work out the exact time in a different timezone. There is an option to add clock widgets but those are not always visible while working in different apps.

Personally I found it difficult to calculate exact time when adding events to calendar or setting meetings in Microsoft Teams as they would rather display my local time.

Work Clock requires macOS 10.12 or later.

Work Clock is one-time purchase with all the updates/upgrades forever.
